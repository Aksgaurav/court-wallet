#  Court Management System

A simple **Court Management System (CLI-based)** built using Python and SQLite.

This project allows **clients and advocates** to interact for case handling through a basic request system.

---

##  Features

*  Secure Login System (with 24-hour account lock after multiple failed attempts)
*  Role-based access (Client / Advocate)
*  Clients can add and manage cases
*  Clients can search advocates by specialization
*  Advocates can send requests for cases
*  Clients can view incoming requests
*  Clients can accept advocate requests

---

##  Tech Stack

* Python 
* SQLite
